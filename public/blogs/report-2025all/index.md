# 前言

> 坐屏幕前准备开始复盘2025年时，一时发愣，然后突然觉得非常惶恐，因为想不起来除了最近几个月之前的事。

这一年发现，越是成长，就越是练就了一身“应对生活”的本事。能旷就旷的课、随便做的作业报告，与人交流就正常对话保持分寸，以前会有些手忙脚乱的突发情况，都能平静妥善的应对过去。但是这样得心应手，好像让情绪滑进了没有波澜的平流层。让我回顾不起来这一年的日子，回想了一下记忆中清晰一点的日子，首次去广东比赛的紧张和激动、温泉约会开心、对某事复盘后的不甘、以前愤怒的、难堪的、愧疚的种种。

情绪是一个锚点，这个锚点会让我更深刻的记起某事，然后通过那件事延展开，就能大概记起那一段光阴。可阅历增长总会让人习以为常，情绪起伏也更低。借用《葬送的芙莉莲》的主题之一：

# 小项目和个人网站

在四月底，突然接到职高的老师联系，做一个德育评价app的ui和前端开发。虽然在校期间帮学校做过招生、报修那些系统的界面，但以前都是天哥安排的，这次是梅姐联系，懵了一波。后面发现还是和天哥一块做，主打一个安心。

![](https://gitee.com/IFILOSEITALL/img/raw/master/20251231172632472.webp)

天哥拉了个群，这个项目的领导也在，狠狠的从天哥那学到了“干活要让领导看见”，不管有什么话都在群里说，不私下问，然后多讲专业术语。从4.26 - 5.5共历时9天，小赚2k+（现在工作了才觉得以前的项目好简单幸福ToT）

# 暑假

打英雄联盟、太热了不想出去玩。也把个人网站ui做完了。

参加了某党支部的团建，去了建川博物馆。小小的攻击到了我乡下人的见识，原来博物馆可以不是一座馆，还可以是一群馆和场地在一起，但是只逛了不到一半。

然后没过几天又是另外一个党支部，包车去了一个景点，一群人只逛了三分之一，拍了点团建照就结束了。太浪费经费了，不是很理解，但上面要求党支部搞党建活动，但确实又有年龄比较大的退伍军人，去这些景点确实逛不完。也许有更好的安排？

见了一个央视等新闻报道过的“善人”，有些虚伪和爱装，又喜欢搞酒桌文化那一套。但论迹不论心吧，确实在做好事，即使用于和别人吹嘘也挺好。

# 开学和面试

大四了来的好快，真吓人。开学旷课去图书馆学习，差不多一个月学会vue了。半个多月把个人网站用vue重写，部署这些搞通。这段时间打英雄联盟发现单双和灵活都无限逼近铂金，但我上不去！终于知道了我很菜，降低游戏频率了。弄完万网站回家去高中见了天哥，给我提了不少建议。开始投简历，同时和网吧里刷新的n个熟面孔开始长达一周的通宵打游戏。第二周的周一早上通宵出来发现有hr联系了，打起精神回答了问题聊了一会。因为成都有两家有意向，加上我在成都有住处，就收拾东西出发。后面就是面试，入职了。

# 总结

爬山的篇幅更大，后面觉得都写很详细也不好。虽然翻相册还是想起来了不少事情，但今年还是很平淡的一年。大概和前言说的一样，太得心应手的日常会掀不起什么波澜。想起了在职高时一个兄弟赛项的队伍里认识了一个很厉害的女生，到现在都觉得她的签名很有帮助。

![](https://gitee.com/IFILOSEITALL/img/raw/master/20251231153005261.webp)

所以新一年会试着跳出舒适圈，虽然需求不是 “最快”的改变自己，但想试着打破常规日常。害怕的事可能是坐脱口秀的前排（暂时不会坐，因为此前从未看过，且钱包有限，这个后面如果看得多的话会试试），总之核心是想试着接触那些曾犹豫的事，让日常的生活有些起伏。

后续会写月报，虽然这次大概想起来了不少事，但应该还是有很多遗漏的，忘记自己的过往这种事还是太残酷了。
