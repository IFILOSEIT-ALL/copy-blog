# 体验高峰期通勤

照常上班的时间出门，早高峰的地铁很挤，换乘时被上车的人群挤到下不了车，等到车在某三街站一下清空了后才能下的去。所以坐过头，回退两站才换上机场线。到达机场、登机、落地、出机场打车到酒店，一路耗时，直到下午四点才到酒店。
然后公司前辈在调试环境和设备，我实习生便没走过流程，大多时间在旁看看，把自己的环境搭建好。有些紧张的还是，主要来自生疏，怕明天工作时出错。

# 工作

第二天早去保密局测试，距离就几分钟路程吧，挺冷的，像小时候通宵上网出来回学校的路上一样冷。保密局的也不是想象中的间谍特务之类的。只是办公楼，门禁多、每个通道和电梯都要刷卡，拍照被禁止，食堂也很普通，价格不算便宜。我一整天几乎没多少事做，上午还蛮紧张吧，但一直没事情，下午就在电脑上玩小游戏了、等他们跑环境测试。临近晚上测试完有一点小需求，花了十来分钟修改代码；然后就结束了，回酒店的路上有点恍惚，可能是在这种机构呆一天后的麻木，又或是我没活要干。

这天晚上改完代码调试后，第三天早再去测试，来测试的小哥很实在，只讲技术，无关其它，没有用权势压人的表现。对这般的人在国家工作又感到一丝庆幸。十一点工作结束。

### 圆明园
最后一天早起抽时间去圆明园，八九点到的，本以为不逗留一小时以内能走完，但园区面积远超预期，只沿着一条路线就花了不少时间。冬天的圆明园挺荒凉感的，植被稀少，长春园像河南那边红薯窖没盖子而已。福海相比倒是独特，湖面已经结冰了，很想走到岛上去，但告示不让踩，留下遗憾。时间紧促，只好匆匆离开。

# 返程
公司买票晚，返程的航班座位选在了J座，左手边是一神采奕奕的老太太，在飞机时上有在写乐谱和歌词，笔迹很干净有力，用的纸有清华的标识，这几天刚好是期末周，可能是清华的某教授老师放假了吧。

起飞前老太太的发卡掉到地上，我在J座，顺手打开手机电筒往缝隙里照，位置很窄，最后还是右边的女生弯腰把卡捡起来。也许正因如此，老太太觉得我们算好人，路上便和我有了几句交流。飞机餐送来时老太太告诉我说她不吃肉，把她的那份里的肉全夹给了我。

后来落地的时候，会竖起大拇指跟我夸机长落地很稳并。然后我有注意到，路人帮她拿行李放到地上时，她会像个小孩般高兴，连声道谢，双手收在胸前举起那种既含蓄又真诚的动作。这样像动漫里的动作在这里倒显得自然，是一种有活力，也有仪式感生活态度。

### 反思

回家路上意识到，我对她的交流几乎只是附和和应答，没能主动把话茬接下去，且老太太把肉都给我之后，我是有意识到老太太可能注重养生这块的，但没有想到把水果给老太太。我自认为平时没有这么木讷，这次原因可能有两方面：一是我是个读民办的，对这种顶尖高校老师，可能有种当时没意识到的自卑在里面；二是我平翘舌不分，有一种交流羞耻心理在作祟。

另一个点是，我和大部分的人在选飞机餐时，因为气流噪音空乘总是需要弯着腰靠近再问一遍。但右边的女生能清楚说出菜名，而不是“麻烦给我来一份xxx吧”这种多余的客套。鉴于前面她帮老太太捡了发卡的印象，我认为她不是那种高冷形象吧。所以无论是想到了这一点刻意为之还是习惯使然，都让我觉得佩服。毕竟空乘在一班航班上要问几十上百个人，如果每个人都能像她那样把意思说清楚，确实能省很多力气。
